# Retail-Orders-ETL-and-Sales-Analytics
End-to-end ETL pipeline using Python, Pandas, MySQL, and SQL for retail sales analytics.

# Project Overview
This project demonstrates an end-to-end ETL pipeline built using Python, Pandas, MySQL, and SQL to process and analyze retail sales data. The pipeline extracts raw order data, performs data cleaning and feature engineering, loads the transformed data into MySQL, and generates business insights using SQL queries.

# Business Problem
Retail organizations generate large volumes of transactional data every day. Raw datasets often contain inconsistent formats and require transformation before meaningful analysis can be performed.

This project automates the ETL process and enables stakeholders to answer important business questions related to revenue, profitability, product performance, and regional sales trends.
